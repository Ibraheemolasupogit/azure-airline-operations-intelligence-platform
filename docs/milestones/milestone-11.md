# Milestone 11

Milestone 11 adds Azure deployment architecture and infrastructure mapping.

## Delivered Scope

- Azure target-state documentation across ingestion, storage, validation, ML, monitoring, GenAI,
  dashboard, security, and governance layers.
- `configs/azure_mapping.yaml` with reference-only policy flags.
- Reference-only Bicep and Terraform skeletons.
- Static architecture safety validation.
- Tests and CI integration for architecture validation.

## Out Of Scope

No Azure resources are provisioned. No Azure SDKs, credentials, tenant IDs, subscription IDs,
Power BI publishing, Fabric publishing, live GenAI calls, deployment workflows, or Milestone 12
portfolio polish are included.
