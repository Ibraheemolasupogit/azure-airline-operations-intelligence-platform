# Dashboard Governance

Dashboard outputs are synthetic, local, deterministic exports for prototype analytics and
evidence review.

They are not live production dashboards, certified airline reports, operational-control tooling,
or decision automation. They must not be used for real passenger, crew, engineering, maintenance,
or airline operations decisions.

The workflow verifies source manifests and checksums, records accepted inputs, rejects
incompatible lineage, writes quality evidence, and publishes local-only responsible-use
disclaimers. No credentials, tenant IDs, workspace IDs, Power BI APIs, Fabric APIs, Azure APIs,
OpenAI APIs, or external services are used.
