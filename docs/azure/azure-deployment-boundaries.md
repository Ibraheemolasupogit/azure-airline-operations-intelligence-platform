# Azure Deployment Boundaries

Milestone 11 provides architecture mapping and reference skeletons only.

## Allowed In This Repository

- Static documentation.
- Mermaid diagrams.
- Placeholder reference templates.
- Static validation of architecture files.
- Local CI checks with no Azure login or credentials.

## Not Implemented

- Cloud resource creation.
- Azure SDK clients.
- Power BI or Fabric publishing.
- Live Azure OpenAI calls.
- Event Hubs runtime clients.
- Production deployment automation.

Any future deployment work must be a separate milestone with security review, environment
approval, secret handling, and operational runbooks.
