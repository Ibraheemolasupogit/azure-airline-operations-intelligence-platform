# Azure GenAI Foundry Mapping

Milestone 9 local assistant behaviour maps to Azure AI Foundry and Azure OpenAI as a future,
governed assistant pattern.

## Target Mapping

- Azure AI Foundry project boundary for assistant configuration and prompt-flow design.
- Azure OpenAI model deployment mapping for future natural-language generation.
- Prompt Flow for grounded retrieval over governed reports, manifests, and dashboard evidence.
- Azure AI Content Safety and deterministic guardrails for response screening.
- Key Vault and managed identity for future runtime access.
- Prompt audit, transcript, evidence pack, response, and guardrail artefacts retained for review.

Milestone 11 does not add SDKs or live model calls. The current assistant remains deterministic
and local.
